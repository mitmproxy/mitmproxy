There are two approaches to installing the __mitmproxy__ certificate authortity in Windows 7. ## Installing the certificate using the command line
  1. Copy the ~/.mitmproxy/mitmproxy-ca-cert.p12 file to your Windows 7 machine.
  2. Run the following command on the command line: __certutil.exe -importpfx mitmproxy-ca-cert.p12__
## Installing the certificate through the Windows User Interface
  1. The easiest way to get the certificate to the device is to use the web app. If this fails for some reason, simply copy the ~/.mitmproxy/mitmproxy-ca-cert.p12 file to the target system and double-click it.
